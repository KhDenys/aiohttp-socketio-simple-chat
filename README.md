# Requirements:


```commandline
$ python -V
  python3.7.*
$ pip install pipenv
```


# Setup pyhton requirements

```commandline
$ pipenv install
```

# Project settings
The main settings are saved in .env file

# Run chat server

Start command:
```commandline
$ pipenv run python chat.py
```

and go to the link http://127.0.0.1:8888/
